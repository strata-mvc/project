iProspect Boilerstrap
==================================================

Projet wordpress vanille basé sur [Wordpress Boilerstrap](http://getboilerstrap.com/)

## Prérequis
- Solution vanille réutilisable
- 100% Responsive 
- Suivant la checklist QA

## Included Librairies 
- Font-Awesome
- Twitter Bootstrap 3
- Snap.js
- GASP TweenMax
- jQuery
- Modernizr

## Bundled Plugins
- AddThis
- Advanced Custom Fields
- Disable Comments
- Simple Page Ordering
- WordPress SEO
- WPML

## Node
- NPM
- Grunt
